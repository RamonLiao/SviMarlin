# 品牌名稱建議

核心概念：比較不同市場的隱含波動率曲面，捕捉價差並可搭配永續合約避險。

1. **Volray** — Volatility＋manta ray，短而有速度感，適合量化交易機器人。
2. **SkewMarlin** — Skew 對應波動率偏斜，旗魚象徵快速捕捉跨市場機會。
3. **ArbCurrent** — 在不同交易場所的價格水流間尋找套利落差。
4. **VegaTide** — Vega 是波動率敏感度，Tide 對應曲面週期與市場潮勢。
5. **SmileEel** — 波動率微笑＋鰻魚曲線，形象化且容易形成視覺識別。
